Simple crosposting from WordPress to the social network Odnoklassniki.

== Description ==

Простой кроспостинг из WordPress в группу социальной сети Одноклассники.

* Автопостинг текста записи или ссылки на запись в группу.
* Простая настройка.
* Подробная инструкция по настройке.
* Абсолютно бесплатный плагин.

Плагин создан на основе плагина VK Poster Group (Djo)

== Installation ==

1. Распакуйте и загрузите содержимое архива в каталог `/wp-content/plugins`
2. Активируйте плагин в разделе 'Плагины' который находится в меню WordPress
3. Произведите настройки плагина в 'Настройки' > OkPosterGP
